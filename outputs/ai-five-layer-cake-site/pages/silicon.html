Fundamentals

经典基础知识筑基

摩尔定律与 Dennard Scaling 失效

晶体管数量仍可增加，但电压和功耗不再同步下降，AI 芯片竞争转向架构、封装、并行ism、数据搬运和系统级能效。

冯·诺依曼瓶颈与存储墙

AI 训练/推理不是只缺 FLOPS，而是大量时间耗在数据从 、SRAM、PCIe/、网络之间搬运。

刷新与 堆叠

需要刷新，传统 DIMM 带宽不足。 通过 垂直堆叠和超宽总线把显存放到 封装旁边。

Bottlenecks

卡脖子瓶颈

绝对硬瓶颈

  • HBM 产能与良率：不是所有 DRAM 产能都能切换成高良率 HBM。
  • CoWoS / 先进封装产能：GPU 逻辑芯片做好后，还必须排队封装才能交付。
  • ABF 基板与测试：大尺寸封装对基板、翘曲控制和测试时间要求极高。
  • 光模块 / SerDes / Retimer：网络不是附属品，而是 AI 集群扩展性的关键。
  • 功耗与散热：TDP 上升后，芯片性能最终受电源和液冷系统约束。

战略瓶颈

  • CUDA 与软件生态迁移成本：硬件替代者最难跨越的软件壁垒。
  • 客户认证周期：HBM、ASIC、服务器平台都要经过超大云客户验证。
  • 资本开支节奏：先进制程、封装、HBM、EUV 都是超重资产扩张。
  • 地缘政治：先进制程、EUV、HBM、出口管制影响全球 AI 算力分布。

Interface Friction

芯片层与上下游的接口摩擦

芯片层的摩擦集中在“模型需求变化太快、半导体供给太慢、系统集成太复杂”。单点芯片性能不再等于可售算力。

模型架构 vs 硬件路线

、长上下文和 MoE 会改变显存容量、、互联与 batch 形态，硬件 tape-out 周期却以年计。

GPU 逻辑 die vs HBM/封装

先进制程产出不等于 AI 芯片产出。、、、known good die 和测试任何一个环节掉速，都会拖慢整卡交付。

芯片峰值算力 vs 集群可用吞吐

训练侧真正看 ，推理侧真正看 TTFT、P99 与每 token 成本。芯片厂必须把网络、驱动、编译器、runtime 和整机系统一起交付。

自研 ASIC vs 软件生态迁移

云厂商自研 ASIC 能压低推理成本，但会牺牲通用性、开发者生态和模型快速迁移能力。 迁移成本仍是最强软壁垒。

CAPEX / Value Capture

资本开支与价值捕获

芯片层的价值捕获从 die 面积扩展到 HBM、先进封装、互联、软件栈和机柜级系统，利润池沿着“不可替代的瓶颈”分配。

核心 CAPEX 变量

  • 先进制程 wafer：N3/N2/A16 等节点的 wafer 价格、良率和客户排产。
  • HBM 配套：HBM4 容量、12Hi/16Hi 堆叠、功耗、良率和客户认证。
  • 先进封装：CoWoS-L/R、SoIC、interposer 尺寸、基板与测试时间。
  • 系统物料：SuperNIC、switch、retimer、光模块、电源、液冷和整机柜集成。

价值捕获位置

  • 平台型芯片商：通过 GPU + 网络 + 软件 + 整柜绑定捕获系统毛利。
  • Foundry / HBM / 封装：用稀缺产能获得更强定价和客户预付款。
  • 高速互联链：以太网/InfiniBand、SerDes、CPO、光模块受益于集群扩展。
  • EDA/IP 与设备：先进节点和先进封装复杂度越高，设计验证和工艺设备越不可替代。
