Industry Status

当前行业现状

训练链正在从单一 GPU 采购竞争，变成全栈工程能力竞争：谁能更快拿到电、更稳定地跑万卡/十万卡作业、更低成本地把 token 转化成能力，谁就拥有模型层的主动权。

已核验事实

  • Meta 公开其两个 24,576 GPU 训练集群，并同时验证 与 路线；Meta 称这些集群用于 Llama 3 训练，并分享了硬件、网络、存储、性能和可靠性经验。
  • Meta 2026 年披露 Prometheus 将走向 1GW 容量，并用 backend aggregation 连接跨数据中心 GPU，说明训练链正在进入 GW 级区域集群时代。
  • NVIDIA GB200 NVL72 是液冷 rack-scale 系统，官方称其连接 36 个 Grace CPU 与 72 个 Blackwell GPU，并形成 72-GPU 域。
  • OpenAI 宣布 Stargate 项目，计划未来四年投资 5000 亿美元建设 AI 基础设施，初始部署 1000 亿美元，训练链的资本开支规模已经接近国家级工业工程。
  • xAI Colossus 官方页面称其已达到 200K GPUs，并披露存储容量超过 1EB、每服务器网络带宽 3.6Tbps、总内存带宽 194PB/s；2026 年 xAI 还称 Colossus 1 拥有超过 220,000 NVIDIA GPUs。

Interface Friction

2026 年连接处核心痛点

硬件-系统接口

  • 集体通信开销：GPU 数量增加后，通信等待会吞掉理论算力，训练吞吐由网络拓扑、拥塞控制和通信库共同决定。
  • HBM/CoWoS 供给：晶圆产出不等于 AI 加速器出货，HBM、先进封装、基板和测试任何一环卡住都会传导到训练排期。
  • Checkpoint I/O：模型越大，checkpoint 越重；写入、复制、恢复和一致性会冲击存储系统与训练效率。
  • MTBF 被规模放大：十万卡集群中故障是常态，训练系统必须围绕硬件故障、silent data corruption 和自动恢复设计。

数据-模型接口

  • 高质量 token 稀缺：公开文本边际价值下降，代码、数学、多模态、企业数据和人类偏好数据更稀缺。
  • 数据配比不可迁移：同样 token 数，不同 mixture 会导致完全不同能力曲线，数据配方本身成为机密资产。
  • 合成数据污染： 能提升特定能力，也可能造成模型近亲繁殖、偏见放大和 benchmark 过拟合。
  • 后训练与评测闭环：训练链末端越来越依赖可复现实验、动态评测和真实用户反馈，而不只是 pretraining loss。

CAPEX & Value Capture

长期价值捕获与资本开支变量

训练链的钱主要沉淀在供给最稀缺、转换成本最高、客户最难替换的节点。

1. GPU/HBM/先进封装

NVIDIA、TSMC、SK hynix、Micron、Samsung 捕获最厚价值。护城河来自架构、软件生态、HBM 客户认证、CoWoS 产能和多年资本开支节奏。

2. 网络与光互联

NVIDIA Networking、Broadcom、Marvell、Arista、光模块和硅光供应链受益于训练集群从 24K、100K 走向 GW 级区域互联。

3. AI 数据中心与电力

电力接入、PPA、液冷、变电站、UPS、CDU 和数据中心 EPC 变成训练链的硬约束。长期合同与地理卡位会形成资产型护城河。

4. GPU 云与模型实验平台

CoreWeave、Oracle Cloud、AWS、Azure、Google Cloud 等把 GPU 集群包装成可租赁训练产能，价值来自可用卡源、交付速度、SLA 和客户锁定。

5. 数据工程与标注反馈

Scale AI、Surge AI、Snorkel、数据版权方和行业数据持有者捕获模型能力的“燃料价值”。真正稀缺的是可授权、可追踪、可评测的高质量数据。

6. 训练框架与通信软件

PyTorch、Megatron-LM、DeepSpeed、JAX/XLA、NCCL、调度和可观测性工具决定昂贵硬件能否转化为 。
