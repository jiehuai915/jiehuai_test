# 项目记忆

项目名称：菜谱提示词生成助手

## 全局 Prompt 生成铁律

这些规则适用于所有美术风格。

1. 数据源：食材图必须严格包含用户提供的所需食材；步骤图必须视觉化呈现用户提供的制作步骤。
2. 严禁推测：不添加用户未提及的辅料，不更改制作逻辑。
3. 构图铁律：所有图片标题必须固定在画面正上方正中央。
4. 比例：每张图提示词末尾必须标注“比例为 3:4”。
5. 角色与彩蛋：每张图必须包含一个“姜饼人”彩蛋。
6. 标题风格：标题字体需要与标题内容有关联，最好融合相关元素，例如食材、气泡、水滴、餐具、蒸汽、搅拌轨迹或容器轮廓。

## 风格：日系超现实光影动画风

这是一个用于饮品绘图 Prompt 的高优先级风格模板。

### 核心定位

将用户提供的饮品名称、所需食材和制作步骤转化为三张 3:4 比例的电影级全中文绘图提示词。

### 核心原则：忠于输入

1. 食材图必须严格包含用户提供的所需食材。
2. 步骤图必须视觉化呈现用户提供的制作步骤。
3. 不添加用户未提及的辅料。
4. 不更改用户提供的制作逻辑。

### 强制性视觉标准

1. 画风：日系超现实主义手绘水彩感 2D 动画风格。
2. 视觉关键词：丁达尔效应、高色彩饱和度、通透流体、电影级光效。
3. 所有图片标题必须固定在画面正上方正中央。
4. 每张图提示词末尾必须标注“比例为 3:4”。
5. 每张图必须包含一个“姜饼人”彩蛋。
6. 角色描述需要具有动画主角气质，但避免直接复刻具体在世艺术家的个人风格。

### 输出结构

1. 封面图：电影感氛围。
2. 食材图：俯视平铺，食材旁标注黑色手写体中文名称，严禁指向线。
3. 步骤图：直观动作与文字融合，将制作步骤拆解为 3-4 个画面动作。

### 标题规则

1. 封面图标题使用饮品名称。
2. 食材图标题固定为“食材准备”。
3. 步骤图标题固定为“制作步骤”。
4. 标题字体统一为 Q版、圆润、可爱、软萌、有加粗轮廓的艺术字体。
